🌉 BridgePoint
3MTT NextGen Knowledge Showcase 2026 — Digital Inclusion Pillar
Find the Hub. Sign In. Belong.
One app that helps every Nigerian find digital access points near them — and ensures that when a deaf or hard-of-hearing person arrives, the hub communicates back in their language.
The Problem
Gap:Nigerians can't find 3MTT hubs, cybercafés, or WiFi zones
Scale:87M+ offline
Gap:Deaf fellows arrive at hubs and can't communicate with staff
Scale:3M+ deaf Nigerians
Gap:No single tool reaches all of Nigeria's LGAs
Scale:774 LGAs nationwide
Three Modules
📍 Find — Offline-first hub locator. Filter by Free WiFi, Deaf-Friendly, Printing, Government. Works in Pidgin, Yoruba, Hausa, and Igbo. Zero data required.
🤟 Learn — Gamified Nigerian Sign Language lessons for deaf fellows, hub staff, and families. XP, streaks, offline-first.
🌉 Bridge — Real-time sign-to-text via phone camera + TensorFlow Lite AI. Deaf users sign → text for staff. Staff type → sign guidance back. No interpreter. No internet.
Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · TensorFlow Lite (simulated) · Web Speech API · Single file · Under 15MB · Runs on any Android device
